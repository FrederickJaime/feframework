Responsive Grid

Create powerful multi-device layouts quickly and easily with the default 12-column, nest-able grid. If you're familiar with grid systems, you'll feel right at home. If not, you'll learn quickly.

Basics

Start by adding an element with a class of row. This will create a horizontal block to contain vertical columns. Then add divs with a column class within that row. You can use column - the only difference is grammar. Specify the widths of each column with the small-#, medium-#, and large-# classes.

SxmPhoenix Framework mobile-first ( small screens first ). Code for small screens first, and larger devices will inherit those styles. Customize for larger screens as necessary.

Flexbox Utilities

Flexbox makes horizontal and vertical alignment painless, through the CSS properties align-items, align-self, and justify-content. SxmPhoenix Framework includes a handful of classes for these properties, which work with any flexbox-enabled component.

Horizontal Alignment

Horizontal alignment classes are applied to flex parents, when dealing with grid we are referring to the row. Center alignment is the default, but you can use one of these classes to change this: .align-left, .align-right, .align-center, .align-justify, .align-spaced

You might be wondering what the difference between .align-justify and .align-spaced is. A justified grid (justify-content: space-between) evenly distributes the space between each column. The first and last columns pin to the edge of the grid.

A spaced grid (justify-content: space-around) evenly distributes the space around each column. This means there will always be space to the left of the first column, and to the right of the last column.

Vertical Alignment

Vertical alignment can be applied to a flex parent—which will align all the children automatically—or to a flex child, which will align only that element. Stretch alignment is the default. To set parent alignment, use these classes: .align-top, .align-middle, .align-bottom, .align-stretch
Note that with vertical alignment, we use the term "middle" for the midpoint, while with horizontal alignment, we use the term "center". As we can't have two CSS classes with the same name, thus we are using different terms.
